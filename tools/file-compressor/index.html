Videos become .compressed.mp4; other files become .gz. A long video can still land over target — the encoder holds a quality floor rather than crushing it to hit the number.

GitHub's browser upload accepts files up to 25 MiB each. Plain git rejects anything over 100 MiB unless it is tracked with Git LFS.

Video compression runs on your machine

This is the hosted demo, and there is no ffmpeg behind it. Non-video files still compress here — they are gzipped in the browser — but re-encoding a video needs the local server. 

git clone https://github.com/mphuongth/video-tools
cd video-tools
pnpm install
pnpm start
